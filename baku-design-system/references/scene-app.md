# Scene: App / Tool

App 型页面应该像能长时间使用的工作界面，优先信息密度和稳定布局。

## 结构

1. 顶部栏：品牌、当前任务、全局动作。
2. 左侧或顶部导航：模式、筛选、工具集合。
3. 主工作区：列表、画布、表格、编辑器或面板。
4. 详情区：选中项、证据、配置、日志。
5. 状态区：保存状态、错误、运行结果。

## 控件

- 模式用 segmented control。
- 二元设置用 toggle 或 checkbox。
- 数值用 slider、stepper 或 input。
- 选项集合用 menu 或 tabs。
- 工具按钮优先 icon + tooltip。

## 视觉

- 不用大 hero。
- 卡片只用于重复项、面板或 modal，不要卡片套卡片。
- 所有固定格式区域要有稳定尺寸，避免内容变化导致跳动。
