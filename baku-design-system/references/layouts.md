# Layouts

为每个 section 选不同布局，不要整页重复同一种卡片网格。

## 1. Workbench Hero

左侧是品牌名、任务承诺和主要动作，右侧是工作台画面、数据面板或头像。适合首页和个人工具介绍。

## 2. Evidence Stack

用纵向时间线或分层条目展示“输入 -> 判断 -> 输出”。适合解释工作流、排障、研究过程。

## 3. Dense Dashboard

顶部指标条，下面两栏或三栏信息块。适合 App 型页面、运营看板、工具面板。

## 4. Document Split

左侧正文说明，右侧引用、命令、表格或示例输出。适合教程和技术说明。

## 5. Avatar Signature

头像/虚拟形象作为视觉锚点，旁边放一句短标签、身份描述和关键作品链接。适合个人 IP 首屏和作者介绍。

## 6. Card Sequence

按 3:4 或 4:5 卡片做连续叙事，每页只讲一个判断。适合小红书、朋友圈、文章转卡片。

## 7. Risk Matrix

二维矩阵或优先级表，把风险、收益、努力成本放在一起。适合方案评审和决策页。

## 8. Command Strip

一条横向命令/快捷动作区域，配合状态 chips。适合工具型页面和工程工作台。
